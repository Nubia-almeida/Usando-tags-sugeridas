O que é semântica

Voltar

Semântica no seu sentido real da palavra significa o sentido das palavras. Assim, quando falamos em um HTML Semântico nos referimos a deixar nossa página de HTML com uma leitura mais fluida e fácil, utilizando tags de fácil identificação para sua estruturação, melhorando assim tanto a leitura dos mecanismos de busca, quanto assim como a navegabilidade para pessoas portadores de deficiência visual.

O que é acessibilidade?

Voltar

Um html semântico deve ser montado utilizando recursos que facilitam, além dos navegadores e mecanismos de busca, os leitores de tela, que lêem em voz alta cada contéudo da página, até mesmo de sua estruturação, como por exemplo uma tabela.

Entendendo Web Scraping

Voltar

É um programa usado para automatizar processos de coleta, consulta e indexação de dados. Os dados que são coletados são públicos, e são utilizados para diversos fins, desde empregar para negócios como um olhar de comparação como também avaliar a reputação de uma empresa. Hoje o melhor exemplo que temos é o googlebot.

A importância de header, main e footer

Voltar

Header

Em geral a tag header sinaliza o conteúdo de introdução da página, onde haverá logotipo, links de navegação, tag h1 (de título), na parte superior da página.

Main

A tag main serve para delimitar o contéudo principal de uma página. Sendo ela a principal, não é permitido que ela seja uma tag "filha" de outra tag, ou seja, não é permitido que se coloque um tag main dentro de uma tag footer por exemplo.

Footer

A tag footer delimita o rodapé do documento. Dentro dele pode-se ter endereços, links, mapas e direitos autorais.

As tags nav, section e aside

Voltar

Nav

Essa tag define o espaço a ser utilizado para os links de navegação do site, que nesse caso pode ser considerado como uma lista de items a ser acessado, podendo ser de forma "ordenada", ou seja, para acessar o próximo link deve passar pelo primeiro, ou de forma "desordenada", quando o usuário pode navegar pelos links de forma aleatória, de acordo com o necessário.

Exemplo de navegação por ordem:
  1. Introdução
  2. Aula 1
  3. Aula 2
Exemplo de navegação por ordem aleatória:
  • Empresa
  • Clientes
  • Endereço

Section

A utilização da tag section do escopo do documento HTML é usado para definir as áreas destinadas às partes temáticas do documento. Essa tag é mais semântica do que a utilização da tag div, por exemplo, que é muito genérica.

Aside

Em geral a utlização da tag aside significa que o conteúdo englobado dentro dele não está diretamente relacionado com o conteúdo principal da página. Pode ser apenas um contéudo complementar ao destacado no main.

Article, Blockquote e Q

Voltar

Article

O contéudo dentro de uma tag article deve fazer sentido por si própio. Pode-se ter várias tags article dentro de uma mesma página. Exemplos de contéudos dentro de um article são posts de blog e postagens de notícias.

Blockquote

Dentro da tag blockquote é especificado uma seção que é uma citação de outro lugar. Por padrão essa tag já fornece um recuo da lateral da página.

Exemplo de utilização de blockquote:
Indica que o texto incluído é uma longa citação. Normalmente, este é processado visualmente pelo recuo (ver Notas sobre como mudá-lo). A URL para a fonte da citação pode ser dada usando o atributo cite, enquanto uma representação de texto da fonte pode ser dada usando o cite elemento.

A tag q

De forma similar ao blockquote, a tag q serve para especificar uma seção de uma citação de outro lugar, porém diferente do blockquote, a tag q serve para citações curtas.

Exemplo de utilização da tag q: Este elemento destina-se a citações curtas que não requerem marcações de parágrafo.

Diferenças entre figure, figcaption e picture

Voltar

Figure

A tag figure serve para delimitar o container (espaço) destinado a uma imagem/figura, dentro de sua página.

Figcaption

A tag figcaption é utilizada para definir a descrição de uma imagem ou figura que está dentro da tag figure.

Picture

De forma semelhante a tag figure, a tag picture também serve para uma imagem/figura, porém nesse caso deve forncer mais de uma fonte de imagem, permitindo que a imagem se ajuste de acordo com os critérios pré-estabelicidos, como por exemplo tamanho de tela.

O que é SEO?

Voltar

A SEO(Search Engine Optimization), significa otimização de mecanismos de busca. São técnicas que ajudam a uma página de site se destacar perante as demais, para aumentar seu tráfego, ou seja, ajuda esses mecanismosa entender seu contéudo e "apresenta-lo" aos usuários para que eles decidam devem acessá-lo.
